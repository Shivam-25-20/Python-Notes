# 📘 Python Course - Day 9: Strings in Python (Part 1)

Welcome to **Day 9** of the **Python Beginner to Advanced** course!

In this lesson, you'll learn the fundamentals of **Strings** in Python. Strings are one of the most commonly used data types and are essential for storing, processing, and manipulating text. You'll explore string creation, indexing, slicing, methods, formatting, and real-world applications with practical examples.
