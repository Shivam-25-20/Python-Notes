# 📘 Python Course - Day 8: Tuples in Python

Welcome to **Day 8** of the **Python Beginner to Advanced** course!

In this lesson, you'll learn everything about **Tuples** in Python. Tuples are ordered, immutable collections that allow duplicate values. They are commonly used to store fixed data and improve program performance.
